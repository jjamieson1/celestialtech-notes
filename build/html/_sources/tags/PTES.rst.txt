PTES
#####

Description
*************

PTES The Penetration Testing Execution Standard (PTES) can be applied to
all types of penetration tests. It outlines the phases of a penetration
test and how they should be conducted. These are the sections in the
PTES:

1. Pre-engagement Interactions
2. Intelligence Gathering
3. Threat Modeling
4. Vulnerability Analysis
5. Exploitation
6. Post Exploitation
7. Reporting
