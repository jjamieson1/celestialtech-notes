#####
cupp
#####

*************
Description
*************

Common user password profiler. Creates a password list using OSINT
information.

*************
Installation
*************

.. code-block:: console

   git clone https://github.com/Mebus/cupp


**options** -i interactive -l online database

********
Usage
********

.. code-block:: console

   python3 cupp.py -i
