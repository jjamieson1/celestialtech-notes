#######
crunch
#######

*************
Description
*************

A program to create word lists

*************
Installation
*************

.. code-block:: console

   sudo apt install crunch


**options:** -t template % for numbers @ for alpha -o output file

**usage**: crunch {start-length} {end-length} -t {template} -o filename

**examples**:

.. code-block:: console

   crunch 1 4 @@%% -o wordlist
