Installation
***************

.. code-block:: bash

    pip3 install pyShodan

Usage 
*******

.. code-block:: bash

    (htb) jjamieso@fatmex$ shodan -h


init the tool

.. code-block:: bash 

    shodan init <api-key>


Scan the internets for ports listening on 502

.. code-block:: bash

    shodan search port:502
