Description 
*************

Zap also comes bundled with a scanner/spider. 

Usage
********

- Step 1:  Find any link in your history and choose `Attack->Spider` from the right click menu.
- Step 2: click start

Passive Scanner 
*****************
This scan will make requests to the end points and attempt to identify potential security vulnerabilities. 

Active Scanner
***************
This scan will run different attacks and report on weaknesses found. 

Reporting
***********
After all your scans are complete, you can create a report of all the findings with `Report->Generate HTML Report`
