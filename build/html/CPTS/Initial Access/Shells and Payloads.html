Payloads

Type of payloads on Windows

D L L s

libraries that help other programs.

B a t c h

.bat files - automating sysadmin style tasks

V B S

Often used in MS Office scripting now

M S I

Microsoft Installer database. The installer looks for msi to understand the components needed to install. You can use msiexec to execute your payload.

P o w e r s h e l l

This is a shell and scripting language

Windows Shells

CMD

When to use: - older host when PS is not available - Simple interactions - using batch files - Execution policies may effect you

Powershell

When to use:

  • You are utilizing cmdlets or other custom scripts

  • interacting with .NET objects

  • stealthy is a lesser concern

  • planning to interact with cloud based services

  • If you script sets and uses aliases

When on the metepreter prompt, you can type shell and get a local prompt

Linux local shell

When you drop a shell with a user that does not have a shell defined in /etc/passwd ( ie: www-data) you can create one in python (if python is installed) with:

python -c 'import pty; pty.spawn("/bin/sh")'

To run sh:

/bin/sh -i

If Perl is installed:

perl —e 'exec "/bin/sh";'

If Ruby (must be run from a script)

ruby: exec "/bin/sh"

If lua (must be run from a script)

lua: os.execute('/bin/sh')

If awk

awk 'BEGIN {system("/bin/sh")}'

using find:

find / -name nameoffile -exec /bin/awk 'BEGIN {system("/bin/sh")}' \;

Using VIM

vim -c ':!/bin/sh'

Checking what sudo permission a user has:

sudo -l
