App Locker

This application created by Microsoft White/black lists applications. Most commonly cmd.exe and powershell.exe are blocked from running, and or what folders they can write too. Even though these are blocked, sometimes alternate locations can be run %SystemRoot%SysWOW64WindowsPowerShellv1.0powershell.exe or PowerShell_ISE.exe

To examine the policy details we can run:

PS C:\htb> Get-AppLockerPolicy -Effective | select -ExpandProperty RuleCollections
